# WhatsApp Clone
A simple full-stack messaging app using Node, Express, and MongoDB.

### About
Built to practice RESTful CRUD operations.
- **Index:** View all chats
- **New/Create:** Add messages
- **Edit/Update:** Modify text
- **Delete:** Remove messages
